## Features

- System Information
- Thermal Monitoring
- WiFi Information
- Health Status
- Connected Clients
  - Total connected clients
  - 5 GHz / 2.4 GHz client count
  - Average / Best / Weakest RSSI
  - Per-client information
    - MAC Address
    - Association Time
    - RSSI
    - SNR
    - TX/RX Rate
    - PHY Mode
    - MIMO (RxNSS × TxNSS)
    - HT Capability
    - VHT Capability
    - MLO Status
    - Power Save Status
  - Raw `wlanconfig` output for both 5 GHz and 2.4 GHz radios
